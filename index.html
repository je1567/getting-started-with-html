World Wide Web (www) and HTML (Hypertext Markup Language>

World Wide Web uses HTML to connect the webpages. HTML is the heart of the web, its text content, markup (how it looks), references to documents (images and videos), and links to other web pages.

Tags

Tags are used to create the HTML code. Depending on the tag, you will have opening and/or closing tags, some tags we have used so far in module 0 are bold, emphasis, anchor, and linkable tags consisting both of websites or images.
